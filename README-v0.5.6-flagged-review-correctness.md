# LifeGrid v0.5.6 — Flagged Review Correctness
Flagged Items is now derived from current active-calendar records through an explicit issue registry. Only registered temporal validation failures and explicitly marked `legacy-ambiguous` records are visible. Valid all-day, unknown, approximate, timed, start-only, and old canonical records are excluded. Saving, confirmation, and deletion reconcile from committed collections rather than cached rows. Grid includes development-only performance marks for the reported approximately 11-second navigation symptom. v0.5.7 will use traces to target rendering, persistence, chunk, and layout costs.
