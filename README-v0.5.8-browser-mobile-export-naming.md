# LifeGrid v0.5.8

v0.5.8 refines Tasks for narrow viewports and makes JSON backup and ICS calendar downloads descriptive, deterministic, and safe. The visible application version is `v0.5.8`; AI interchange remains v4 and backups remain schema 6.

Downloads use `lifegrid_<descriptor>_<Calendar-Slug>_<YYYY-MM-DD_HH-mm-ss>.<extension>`. Backup remains a complete store backup (all calendars); its active calendar name is user context, not a claim of single-calendar content.
