## Ordering and header

- [ ] Verify status grouping behavior remains explicit
- [ ] Check shared header on Grid, Tasks, Projects, People, AI, Settings; version, calendar clock, UTC, minute update, long name, wrapping and no overflow

## Mobile and recovery
- [ ] Narrow layout, orientation, editor scrolling, keyboard visibility, reorder controls
- [ ] Restore pre-test backup and verify original active calendar

Severity: blocker = data loss/corruption; major = workflow unavailable; minor = presentation/accessibility. Stop on any blocker. Failure report: steps, expected, actual, calendar, browser/device, screenshot URL, console output.
